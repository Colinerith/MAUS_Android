## 주요 기능  
1. Switch Fragment  
 : 스위치 이미지를 터치하여 스위치 on/off를 조절할 수 있다.
2. Timer Fragment  
 : 설정한 날짜/요일/시간에 맞춰 스위치가 on/off 되도록 설정할 수 있다.
    - Add: '+' 버튼을 눌러 타이머를 추가한다. [시간(오전/오후, 시, 분), 날짜(연-월-일), 요일, ON/OFF]
    - Delete
    - Modify
3. Setting Fragment  
    - Calendar
    - User Info
